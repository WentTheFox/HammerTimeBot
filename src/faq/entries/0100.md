# Can you change how the time and/or date is displayed by Discord?
No, the display of the timestamp syntax is **entirely left up to Discord** ([documented here](<https://discord.com/developers/docs/reference#message-formatting-timestamp-styles>)), what HammerTime does is simply provide a convenient way for you to get the code needed to use these pre-existing built-in formats. In simple terms this means I cannot help you:
* change how the timestamp looks in any way, shape, or form
* make it display a specific format regardless of client language settings
* add additional information, like a timezone
* change what is in the tooltip or hide it entirely
If you would like additional formatting options, please [send a feature request to Discord](<https://support.discord.com/hc/en-us/articles/213490807-Where-can-I-submit-Feature-Requests-Feedback-and-Suggestions->).
