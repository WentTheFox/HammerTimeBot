# When I enter a timestamp code in a message it turns into "F" after sending, what's going on?
You are most likely using a 3rd-party client modification called BetterDiscord, which has a plugin called SendTimestamps. Disabling this plugin should allow you to send timestamps correctly. Also, this is a friendly reminder that these modifications are against Discord ToS and by asking this you are outing yourself as someone using such modifications.
