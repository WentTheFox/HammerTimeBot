# Can you read timestamps from sent chat messages and reply with the dynamic timestamp?
This cannot be done automatically as it would require the privileged intents from Discord that enables reading all message contents, which requires separate justification (e.g. moderation purposes) to have access to. Discord ToS specifically disallows using this intent for the purposes of executing automated commands, and I do not intend to apply for more permissions than necessary.
If you want to extract Discord-formatted timestamps from an exising message, like this one: <t:1697994570:R> you can right-click any message and use the _Apps > Extract Timestamps_ context menu command provided by <@964106782790283295> which will find timestamps in the message you right-clicked. Feel free to try it on this message.
