# Can I create a timestamp that always displays my current local time for others?
Due to the limitation mentioned above, this is also not possible. However, there is a common workaround used by many to indicate your time offset to others. This can be done by either:
* using the "short time" format to generate a timestamp using [the website](<https://hammertime.cyou>) or
* the </at:1023559188674256918> command of the <@964106782790283295> bot, setting the hours, minutes, and seconds to 0
This will create a timestamp that points to the midnight of the specific day you did this process on.

Afterwards you can copy the "short time" syntax (the one that looks like `<t:123456789:t>`) and paste it into your bio (or any message) with some additional text around it, for example:
> My midnight is your <t:1731366000:t>
To people in a different timezone this will display when midnight for you is in their own time.

Note that this method has an important caveat: because timestamps indicate a specific point in time, the time displayed will never change based on your current timezone (e.g. daylight savings, traveling, etc.). You will need to re-generate the timestamp with a date that is matching your current timezone manually.
