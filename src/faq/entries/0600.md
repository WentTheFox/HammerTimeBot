This default can be overridden by changing your user settings for <@964106782790283295> by either
1. using the </settings:1231738209369587762> command to view your settings page directly, or
2. by logging in on the <https://hammertime.cyou> website manually.
This will automatically apply to any <@964106782790283295> command ran by your Discord account.
